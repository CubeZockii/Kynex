# Moderation-Bot

A Moderation Bot that can be used for your Server. Has an option to be used as a Logging Bot.

# Features/Commands:

- Ban/Unban
- Kick
- Purge
- Warn/Unwarn
- Warnings
- role
- help
- Mute/Unmute
- Lock/Unlock
- ping
- Logging
- Automatic Message Moderation System
- setup log/error channel
- show current prefix
- change prefix for server
- setup alert channel
