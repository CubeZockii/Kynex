ModBot Documentation

All you need to know to use and host ModBot

ModBot is a versatile moderation bot for Discord servers. You can use it as-is or self-host it.

General Info

The bot includes an optional logging system. If you choose not to use it, you may see a startup error like:

Error loading LoggingCog: Extension 'logging_cog' could not be loaded or found.

This is safe to ignore.

Features / Commands

  • Ban / Unban
  • Kick
  • Purge
  • Warn / Unwarn
  • Warnings overview
  • Automatic punishment system
  • Mute / Unmute
  • Lock / Unlock channels
  • Manage roles
  • Logging (optional)
  • Revive inactive chats
  • Ping & Help commands

Hosting Yourself

You can self-host ModBot by downloading the source files on the HomePage and running them with Python. Make sure to install the required dependencies listed in requirements.txt.
