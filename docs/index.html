Automatic Punishment

Warnings are tracked. If a user hits a threshold, ModBot can automatically mute or ban them. You can customize this logic in the code.

Logging System

If unused, you may see:

Error loading LoggingCog: Extension 'logging_cog' could not be loaded or found.

To remove this, delete the error handling code at lines 60-61.

Self-Hosting

  1. Download ZIP
  2. Extract ZIP & open terminal in folder
  3. pip install -r requirements.txt
  4. Configure token via .json or .env
  5. python bot.py

Recommended hosts: KataBump, SillyDev.
